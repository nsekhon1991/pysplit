Concentration / Utilities / Binary File Inverse

This menu calls the utility program con2inv to compute the inverse of the binary concentration file and write a new binary file. Each grid cell is replaced by the inverse (1/C) of its value (C).



USAGE: con2inv [options]
-i[input file name (cdump)]
-o[output file name (concinv.bin)]
-d[process id (bin)]
