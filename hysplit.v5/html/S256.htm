Trajectory / Special Runs / Clustering / Display

From Step 3. Get Results, Trajectory Cluster Analysis, Display Means, Display Clusters, and Trajectories not used will all open the "Trajectory Cluster Display" window.

The Trajectory Cluster Display window is similar, but not identical, to that used for plotting individual trajectories: Trajectory Display.

The postscript graphics will be converted to GIF if the GIF checkbox is checked. Choose "Frames" at the Postscript Conversion window for the plots of trajectories in each cluster.

The trajectory endpoints will be output in Generate format if the GIS checkbox is checked. To convert to ESRI shapefile, exit the cluster windows, and go to Trajectory, Utility Programs, GIS to Shapefile. This converts one frame at a time.

To set a constant map domain, set Rings: choose the Set checkbox, number of rings: 0, Dist (km): map radius. The time label reference is the trajectory start. The plot trajectory duration defaults to the hours clustered. The vertical coordinate is set to none.
