Concentration / Utilities / Binary File Average

This menu calls the utility program conhavrg to spatially average the HYSPLIT binary concentration file and write a new binary file. Each grid cell is replaced by the average value of the center cell and the surrounding cells specified by the scan radius. The default scan radius of 1 means that 9 grid cells will be averaged together.


USAGE: conhavrg [options]
-i[input file name (cdump)]
-o[output file name (cavrg)]
-s[surrounding grid points to average (1)]
