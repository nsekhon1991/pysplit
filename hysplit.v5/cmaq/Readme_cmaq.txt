DIRECTORY: ./cmaq
______________________________________________________________

Utility programs to convert HYSPLIT output to CMAQ IOAPI
or CMAQ MCIP meteorological data for input by HYSPLIT

traj2api - convert HYSPLIT trajectory endpoint files to ioapi format
conc2api - convert HYSPLIT binary concentration file to ioapi format
mcip2arl - convert CMAQ MCIP meteorological data to ARL packed data
