____________________________________________________________________
The following directories have been installed. 

archive - Default directory to which cluster output files are saved
   when the "Archive" button is pressed.  Contains no files at installation.

endpts - This directory may be used to hold the trajectory endpoints files.
   Contains no files at installation.

example - This directory has a sample set of trajectory endpoints files
   (endpts) and the cluster analysis output (output).

working - Default working directory for trajectory cluster analysis.  
   Contains no files at installation.
