CDF2ARL

cdf2arl converts NCEP/NCAR Reanalysis data in NetCDF4 file
to a HYSPLIT compatible format.

Required library
NetCDF version 4 (use -lnetcdff)

Input files from ftp://ftp.cdc.noaa.gov/Datasets/ncep.reanalysis/
