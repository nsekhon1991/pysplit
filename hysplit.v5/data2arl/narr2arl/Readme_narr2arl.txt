DIRECTORY ./narr2arl

===================================================================
This directory contains the code to convert North American Regional
Reanalysis (narr) model output to a format compatible for input to
HYSPLIT.  The makefile puts the executable in ../../exec.  NARR
archived files in HYSPLIT-format are at 
ftp://ftp.arl.noaa.gov/archives/narr/

----------------------------------
PROGRAM: /exec/narr2arl
 Convert pressure narr grib1 file to arl format
 Usage: narr2arl [file_1] [file_2] [...]
