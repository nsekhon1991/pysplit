DIRECTORY: ./scripts
_________________________________________________________

Sample shell scripts to automate concentration and 
trajectory simulations and the extraction of meteorological
data at a point from the packed data file.

---------------------------------------------------------

autoconc.sh	- simple concentration script
autotraj.sh	- simple trajectory script
multiconc.sh    - complicated multiple concentration
xtrct_stn.sh	- meteorological data extraction
tcm_extract.sh  - process transfer coefficient matrix
